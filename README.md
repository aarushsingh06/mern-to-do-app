MERN To-Do List Application
Overview
This is a simple To-Do List application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). Users can add tasks and view them through a basic and easy-to-use interface.
Features
Add new tasks
View saved tasks
Connect frontend and backend
Store data in MongoDB Atlas
REST API built with Express.js
Technologies Used
React.js
Node.js
Express.js
MongoDB Atlas
Axios
Postman
Project Structure
mern-todo-app/

Installation
Backend Setup
cd backend
npm install
npm start
Frontend Setup
cd frontend
npm install
npm start
